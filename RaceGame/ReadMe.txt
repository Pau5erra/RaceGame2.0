A 3D Race Game

This game consist of to take 10 balls before 20 seconds!

If you take all the balls, then your time is saved for next game.
Who will be the best?

Try it!

GAME:
Instructions:

Title:
Current time, Balls left, best time

Controls:
Arrow UP : Start/Accelerate
Arrow Down: Back
Arrow Left/Right: Direction of car
B : Hand Break

Keys:

R: Restar game

FOR DEVELOPER:
F1: Debug mode
F2: God mode
C: Create a ball
P:Change the cam between car cam and control cam with ASDW UP/DOWN:QE
